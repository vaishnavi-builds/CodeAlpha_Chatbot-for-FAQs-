# FAQ Chatbot using NLP
An interactive FAQ Chatbot that answers user questions by matching them with predefined FAQs using NLP-based text similarity and intent matching. The chatbot features a modern UI and provides the most relevant response based on user input.

## 🚀 Features
* Interactive chat UI
* FAQ-based responses
* NLP text preprocessing
* Intent / keyword matching
* Similarity-based answer selection
* Typing animation
* Timestamp messages
* Responsive design

## 🛠️ Technologies Used
* HTML
* CSS
* JavaScript
* NLP preprocessing
* Similarity matching

## ⚙️ How It Works
1. User enters a question
2. Text is converted to lowercase and cleaned
3. Question is matched with FAQ dataset
4. Best matching question is selected
5. Chatbot returns the corresponding answer

## 💬 Example Questions
You can try asking:

* return policy
* payment methods

* delivery available
* location
* where are you located
* how to return product

## ▶️ How to Run
1. Download or clone this repository
2. Open project folder
3. Double-click `index.html`
